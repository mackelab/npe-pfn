# Paper figures

Figures should be create here. Some of formating constraints are:
- Figures should be in .svg format
- Figures should have a **width of 14cm** (double column) or 7cm (single column)
- Figures should have the correct style i.e. use `use_style("pyloric")` inn python. This will also take care of font size and font family, which should not be changed manually.
- At best most of the figure should be generated by code (depending on figure) and we can still use inkscape or adobe illustrator to make some final adjustments.

Note: basically all experiment configs, including `conf/experiment/sbibm_core.yaml`, override `/partition` and therefore require a matching partition config in `conf/partition/`. This is specific to SLURM environments, so make sure to add your local partition name(s) and resource settings there (for example CPUs, memory, GPUs, and timeout) before running them.
